## About Me

Most software engineers begin their journey through Computer Science.

Mine started in Biochemistry.

While studying at Lagos State University, I developed a deep appreciation for understanding complex systems, analyzing problems methodically, and finding patterns beneath the surface. Those same principles eventually led me into software engineering, where I discovered a passion for building systems that are both functional and impactful.

Today, I build modern web applications using TypeScript, React, Next.js, PostgreSQL, and other technologies across the JavaScript ecosystem. I enjoy working across the entire development lifecycle—from designing databases and authentication systems to creating intuitive user experiences and deploying production-ready applications.

My approach to engineering is simple: stay curious, keep building, and never stop improving.

🌍  I'm based in Nigeria
